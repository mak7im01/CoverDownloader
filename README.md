# CoverDownloader для PulseSync

Аддон для скачивания обложки трека

## Возможности

* Кнопка скачивания на обложке в полноэкранном плеере (левый верхний угол, появляется при наведении)
* Кнопка скачивания (inline-кнопка) рядом с названием трека — скачивание без открытия полноэкранного режима
* Если установлен аддон CopyTrackName — настраиваемое положение кнопки относительно иконки копирования
* Скачивание обложек одним кликом
* Настройка качества изображений
* Настройка формата имени файлов
* Настройка размера и прозрачности inline-иконки
* Уведомления о скачивании

## Настройки

### Основные настройки

**Включить кнопку на обложке**
* Включает или отключает кнопку скачивания на обложке в полноэкранном плеере
* По умолчанию: включено

**Качество изображения**
* Низкое (200x200) - для быстрого скачивания
* Среднее (400x400) - оптимальный баланс
* Высокое (1000x1000) - отличное качество
* Оригинал (максимум) - оригинальный размер без сжатия
* По умолчанию: высокое

**Показывать уведомления**
* Отображает уведомление при успешном скачивании
* По умолчанию: включено

### Сохранение файлов

**Формат имени файла**
* Настройте шаблон имени файла
* Доступные переменные:
  * `{artist}` - имя исполнителя
  * `{title}` - название трека
* По умолчанию: `{artist} - {title}`

### Внешний вид (inline-кнопка)

**Включить inline-кнопку**
* Включает или отключает кнопку скачивания рядом с названием трека
* По умолчанию: включено

**Позиция иконки (при активном CopyTrackName)**
* Справа от иконки копирования — по умолчанию
* Слева от иконки копирования
* Не влияет если аддон CopyTrackName не установлен

**Размер иконки**
* Размер иконки скачивания рядом с названием трека (12–24 пикселя)
* По умолчанию: 18

**Прозрачность иконки**
* Прозрачность иконки в неактивном состоянии (5–100%)
* По умолчанию: 70%
